# AI-MD single-cell RNA-seq
## Requirements

### Tools

- CrossMap (v0.6.1)
- vcftools (v0.1.16)
- bcftools (v1.11)
- popscle, demuxlet
- Cellranger - 6.1.2 (October 25, 2021)

### Datasets

1. GEX
   - R1, R2, I1, I2
2. TCR
   - R1, R2, I1, I2
3. BCR
   - R1, R2, I1, I2
4. GSAv2
   - SNPs
