# AI-MD single-cell RNA-seq
## Requirements

**Tools**

- crossmap
- vcftools
- bcftools
- popscle, demuxlet
- Cellranger - 6.1.2 (October 25, 2021)

**Datasets**

1. GEX
   - R1, R2, I1, I2
2. TCR
   - R1, R2, I1, I2
3. BCR
   - R1, R2, I1, I2
4. GSAv2
   - SNPs
